================================================================
  飞书风格 HTML Deck  ·  使用说明
================================================================

这个 zip 包里有什么
─────────────────────
  index.html        ← deck 本体（双击在浏览器打开），内置可视化编辑器
  assets/           ← 字体 / 图片 / logo 等资源（别删，别改名）
  deck.json         ← (如附带) deck 的源数据，留给生成方做结构性修改
  README.txt        ← 你正在看的这份说明


第一次使用：浏览 deck
─────────────────────
  1. 双击 index.html
  2. 在浏览器里：
       ←/→ 或空格    ── 翻页
       Esc           ── 退出全屏 / 退出编辑模式
       ⌘/Ctrl + F    ── 全屏


改文字（不动布局）—— 直接在浏览器里改
─────────────────────
  这份 deck 自带一个零依赖的可视化编辑器，不需要装任何东西。

  1. 双击 index.html 打开
  2. 按 E 进入编辑模式
  3. 点任意一段文字，直接改（像改文档一样）
       · 想调整页序：拖动左侧的页面缩略图即可
  4. 按 ⌘/Ctrl + S 保存
       · Chrome / Edge：第一次会弹一个授权框，再弹系统选择框，
         选中当前这个 index.html（macOS 上系统按钮可能写「打开」），
         之后每次保存都会静默原地覆盖。
       · Safari / Firefox：会下载一个新的副本，用它替换原文件即可。
  5. 按 Esc 退出编辑模式


改样式 / 颜色 / 排版
─────────────────────
  这一类在浏览器编辑器里改不了（它只管文字和页序）。
  样式调整请联系最初生成这份 deck 的人，让他改 CSS 后重新出包。


改结构 / 加新页 / 改 layout
─────────────────────
  回到生成这份 deck 的会话里说一句，例如：「给 slide 5 之后加一页 stats，
  内容是…」，让 Claude / OpenClaw 改 deck.json 后重新生成即可。


技术细节（FYI）
─────────────────────
  - 编辑器是纯客户端 JS，离线可用，file:// 或 https:// 都行。
  - Chrome/Edge 用 File System Access API 原地保存；其它浏览器走下载兜底。
  - 编辑器只改可见文本叶子和页序，不动 CSS / SVG / 装饰元素。
  - 如需把改动沉淀回源头，请把改后的 index.html（或 deck.json）发回生成方。
