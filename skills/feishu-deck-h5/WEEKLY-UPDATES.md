## 2026-05-30 · 本周更新(覆盖 5/23 – 5/30)

> 这套**按飞书品牌规范一键生成 HTML 汇报 deck** 的技能,本周一批更新——**出品更丰富、更稳、更快**。重点如下 👇

### 🆕 新能力
- **数据图表上线**:现在能直接生成 **柱状图 / 折线图 / 环形图**——给数字就行,布局自动算、配色锁定飞书品牌色。以前数据页只能堆文字卡或 KPI 数字,现在能真正「图表化」。
- **整页搬运(Native Lift)**:从别的 deck 直接把一页搬进你当前的 deck,**图片 / 原型 / 字体等资产一起自动带过来**,不用手动剪、不会搬完图裂。
- **一键换皮(Reskin)**:把外来的 HTML 页面机械套成飞书风格(配色/字体/版式),保留原设计、只换「皮」。

### 🎨 出品质量更稳、更符合预期
- **设计先行默认开**:生成前先出设计方案、逐页判断「要不要配图标/配图」、并做「这页装得下吗」的密度检查——直击「生成效果 ≠ 预期」。
- **图标库扩到 43 个**(原来 8 个),纯文字页更容易配上合适的图标。
- **新增一批「肉眼难发现」的版式自检**:自动提醒「整 deck 太平没图」「视觉没重点 / 信息平铺」「上空 / 下空 / 中空不均衡」「中文末行孤字」等。
- **误报修复**:校验不再把图表坐标轴标签(HIGH/LOW)、含年份的英文标题(如 *ANNUAL REVIEW 2025*)误判成「中英混排」乱报警——少了告警疲劳。
- **带视频的页**:进场自动重播 + 自动开声。

### ⚡ 更快更省
- **技能瘦身 83%**(420KB → 71KB):每次生成占用的上下文砍掉一半多,**更快、更省**,甚至用较小的模型也能跑出好效果。
- **长 deck 手机浏览更流畅**(只渲染当前可见页)。

### 🔧 底层稳定性大幅加固(你不用关心,但意味着以后更靠谱)
- 自动化测试从 ~16 条扩到 **138 条**,核心规则全有回归守护。
- 校验规则做了「单一真相源 + 模块化」,文档与代码自动对齐、不再各处手抄漂移。
- 仓库卫生 + 多项 bug 修复(编辑模式虚线框残留、打字误触快捷键、交付打包等)。

> 持续打磨生成效果,有问题随时反馈 🙌
